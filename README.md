# 🏥 Healthcare Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Healthcare Analytics Dashboard developed using Microsoft Power BI. It provides insights into patient admissions, billing, hospitals, doctors, insurance providers, and medical conditions.

---

## 📊 Dashboard Features

- KPI Cards
  - Total Patients
  - Total Billing
  - Total Hospitals
  - Total Doctors
  - Average Billing
  - Average Stay

- Monthly Patient Admissions

- Medical Condition Analysis

- Top 10 Hospitals by Billing

- Top 10 Doctors by Patients

- Insurance Provider Distribution

- Gender Distribution

- Blood Type Analysis

- Admission Type Analysis

- Interactive Slicers

---

## 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX
